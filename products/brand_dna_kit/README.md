# Brand DNA Kit

**Version:** 1.0 · **Platform:** Gumroad · **Price test points:** $29 / $39 / $59 / $79

## What this is

A template + discovery script + 3 worked examples for extracting a
brand's DNA (voice, audience, objections, anti-avatar, value pillars)
in a single 90-minute session. The output is a single-page profile you
can pin to a wall and calibrate every piece of content against.

## Contents

- `BRAND_DNA_GUIDE.md` — the methodology, 4-ring style
- `DISCOVERY_SCRIPT.md` — the 6-bucket intake with verbatim prompts
- `TEMPLATE_BRAND_DNA_PROFILE.md` — the 1-page output template
- `examples/epos_brand_dna.md` — completed profile for EPOS (live reference)
- `examples/consultant_example.md` — completed profile for a composite consultant
- `examples/local_service_example.md` — completed profile for a local service business
- `BUILD_PDF.sh` — PDF generation (pandoc)

## Who this is for

Founders who can't answer "what makes you different?" in one sentence.
Consultants whose proposals sound like their competitors'. Service-
business owners who feel like they're shouting into the same crowd
everyone else is.

## Who this is NOT for

Enterprise brand teams running six-figure rebrands. This is a solo /
small-team discipline. If you already have an approved brand book, this
kit adds nothing.

## Build the PDFs

```bash
bash BUILD_PDF.sh
```
